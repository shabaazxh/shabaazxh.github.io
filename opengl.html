Vulkan Renderer (2025)

Vulkan

This is a toy Vulkan renderer, written to explore various real-time rendering techniques. A detiled report which provides a complete breakdown of the current renderers features can be found here.

Features implemented

  • Physically Based Shading (PBR)
  • Bloom
  • Horizon-Based Ambient Occlusion (HBAO)
  • Screen-Space Reflections (SSR)
  • Shadow mapping with PCF
  • Debug visuals: Mesh density, overdraw and overshading
  • Deferred and Forward

Available on Github

Budget FIFA (2020)

OpenGL

Budget FIFA is an incredibly immersive Football experience on an extremely tight budget (Just kidding). The goal in this game is to shoot the ball towards as many score points as you can to collect the most amount of points. Shoot as many targets as you can to get the most amount of score before your attempts are depleted.

Features implemented

  • Target Collision detection
  • Game Skybox
  • Shooting the ball
  • Updating Score/Attemps
  • Rendering 3D objects
  • Rendering a 2D projection ontop of 3D projection
  • Scene Lighting
  • Game Textures
  • Keyboard Input

Available on Github

Escape The Room (2020)

OpenGL

Escape The Room is a game that requires the player to avoid the obstacles in their path to reach the other side of the room to reach the next level. Colliding with the obstacles will reduce the players point and take them back to the beginning. Your goal is to escape the room without hitting any obstacles, completing as many levels as you can to gain the most amount of points.

Features implemented

  • 2D Box Collision detection (AABB)
  • Player movement (Arrow keys)
  • Player rotation
  • Grid ON and OFF
  • Score/Points system
  • Keyboard interaction
  • Mouse Interaction (right-click to enable features)

Available on Github
