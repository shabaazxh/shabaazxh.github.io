Escape The Room

OpenGL

Escape The Room is a game that requires the player to avoid the obstacles in their path to reach the other side of the room to reach the next level. Colliding with the obstacles will reduce the players point and take them back to the beginning. Your goal is to escape the room without hitting any obstacles, completing as many levels as you can to gain the most amount of points.

Features implemented

  • 2D Box Collision detection (AABB)
  • Player movement (Arrow keys)
  • Player rotation
  • Grid ON and OFF
  • Score/Points system
  • Keyboard interaction
  • Mouse Interaction (right-click to enable features)

Available on Github

Budget FIFA

OpenGL

Budget FIFA is an incredibly immersive Football experience on an extremely tight budget (Just kidding). The goal in this game is to shoot the ball towards as many score points as you can to collect the most amount of points. Shoot as many targets as you can to get the most amount of score before your attempts are depleted.

Features implemented

  • Target Collision detection
  • Game Skybox
  • Shooting the ball
  • Updating Score/Attemps
  • Rendering 3D objects
  • Rendering a 2D projection ontop of 3D projection
  • Scene Lighting
  • Game Textures
  • Keyboard Input

Available on Github

The Saga Begins

Vulkan

This has been my first step into exploring and trying to conquer the Vulkan API. Vulkan has always been a graphics API I have been keen to learn and understand since it was the more modern way of doing computer graphics. While, Vulkan did present itself to be more complicated than OpenGL, after some exploration of how the API worked I managed to put this small project together which aimed to have a coloured background, camera movement and also rendering a 3D object that was spinning.

Instanced rendering with Vulkan

Vulkan

Rendering multiple objects using Vulkan's instancing method

Experimenting with Instanced rendering in Vulkan. Re-using and executing the same drawing commands to render multiple instances of the same object, resulting in large amounts of geometry with fewer API calls

Features in demo

  • Instanced Rendering
  • Ambient lighting
  • Diffuse lighting
  • Specular lighting
  • Texture loading/rendering
