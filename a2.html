ProjectX - A Vulkan Renderer

Abstract

This project showcases a Vulkan renderer focused on real-time rendering with rasterisation. It implements both forward and naive deferred shading pipelines, exploring challenges such as overdraw, overshading and mesh density. Additionally, it explore a compressed TBN matrix compression strategy for normal mapping with minimal memory overhead. The renderer features shadow mapping with percentage-closer filtering (PCF), Physically Based Shading (PBR), and Bloom. It has since been extended with Screen-Space Ambient Occlusion (SSAO) specificially HBAO and Screen-Space Reflections (SSR). Source code for the project can be found on Github.
