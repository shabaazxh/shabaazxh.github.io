K-Means & MLP

This page is still under development

In the first part of this report, I perform K-means clustering on a vehicle dataset to find natural groupings (unsupervised learning) among the vehicle data in the dataset. The second part of the report introduces a time-series problem where I use a Neural Network, experimenting with a single hidden layer, a Neural Network with 2 hidden layers, different input values and also experimenting with various algorithms such as 'backprop', 'rprop+', and 'rprop-'. I will evaluate the performance of each of these Neural Networks using statisical indicies RMSE, MAE & MAPE. The best single hidden and 2 hidden layer Neural Network will be discussed from which we will finally conclude which one out of the two approaches is most desirable.

Each part of the report will introduce important techniques used in machine learning and some specific techniques used when performing the task at hand, providing relevant code examples of these techniques being used and the motivation behind using these techniques.

All code demonstrated uses the R programming language.
